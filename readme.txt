A simple video gallery plugin for WordPress.

== Description ==

This plugin allows you to create a simple modal video gallery with support for YouTube and Vimeo videos.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/lightbox-video-gallery` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the provided shortcode `[video_gallery]` to display the video gallery or [video_gallery category="category-name"] for specific category.

== Frequently Asked Questions ==

= How do I add videos to the gallery? =

Add a new video post through the 'Videos' custom post type and enter the video URL in the provided meta box.
